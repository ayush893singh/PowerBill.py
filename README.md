## PowerBill

A simple Python project that calculates electricity bills based on the number of units consumed.

---

## Features

- Calculate electricity bill instantly
- Uses slab-based billing system
- Beginner-friendly Python project
- Simple command-line interface

---

## Technologies Used

- Python

---

## Concepts Used

This project uses:

- Variables
- User Input
- Arithmetic Operations
- Conditional Statements (if, elif, else)
- Output Formatting

---

## Billing Rates

Units Consumed| Rate
0 – 100| ₹5 per unit
101 – 200| ₹7 per unit
Above 200| ₹10 per unit

---

## Installation

1. Install Python from https://www.python.org/
2. Download or clone this repository.
3. Open a terminal in the project folder.

---

## How It Works

1. User enters the number of electricity units consumed.
2. The program checks which billing slab applies.
3. The bill amount is calculated.
4. The final bill is displayed.

---

## Example Output

===== Electricity Bill Calculator =====

Enter electricity units consumed: 250

Total Bill Amount = ₹1700.0

---

Calculation Example

For 250 units:

- First 100 units → 100 × ₹5 = ₹500
- Next 100 units → 100 × ₹7 = ₹700
- Remaining 50 units → 50 × ₹10 = ₹500

Total Bill = ₹1700
